# Faith Ahebwa - Software Developer Portfolio

A modern, responsive, and animated portfolio website showcasing my work, skills, and experience as a software developer.

## Features

✨ **Modern Design**
- Clean, professional layout with dark theme
- Smooth animations and scroll interactions
- Fully responsive design (mobile, tablet, desktop)

🎯 **Sections**
- **Hero** - Eye-catching introduction with profile image
- **About** - Personal bio and professional info
- **Skills** - Tech stack with progress bars
- **Projects** - Portfolio of completed work
- **Contact** - Direct email integration
- **Footer** - Centered branding

⚡ **Interactive Elements**
- Scroll-triggered animations
- Hover effects on cards and buttons
- Form validation
- Parallax effects
- Smooth navigation links

## Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Fonts**: Google Fonts (Bebas Neue, DM Sans)
- **Animations**: CSS Keyframes & Intersection Observer API
- **Contact**: Mailto integration (opens email client)

## How to Use

### View the Portfolio
1. Open `index.html` in any modern web browser
2. Scroll through sections to explore work and skills
3. Click nav links for smooth scrolling navigation

### Contact Form
1. Fill in your name, email, and message
2. Click "Send message →"
3. Your default email client opens with the message prefilled
4. Send the email to receive your message

### Customize Content

**Edit your information:**
- Update name, title, and bio in HTML
- Change skill percentages in skill bars
- Modify project descriptions and tags
- Update contact links (email, LinkedIn, GitHub)
- Add/remove sections as needed

**Update profile image:**
- Replace `WhatsApp Image...jpeg` with your own photo
- Update the filename in the `<img>` tag in index.html

**Adjust styling:**
- Edit `styles.css` for colors, fonts, spacing
- Modify media query breakpoints (968px, 640px, 480px)
- Customize animation speeds and delays

## Responsive Breakpoints

- **Desktop**: Full layout at 968px+
- **Tablet**: 640px - 967px
- **Mobile**: Below 640px and 480px optimized views

## Animations

- **fadeInUp** - Elements slide up with fade
- **fadeInLeft** - Elements slide from left
- **fadeInRight** - Elements slide from right
- **float** - Subtle floating motion
- **slideDown** - Navigation slides down on load
